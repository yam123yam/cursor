Import from CSV
The CSV should have the following columns in order:
  • First Name
  • Last Name
  • Grade
  • Team (Varsity, JV, A, or Dev Squad)
  • Workout Group (last column)

Workout Group Format:

  • The last column is always the workout group
  • Athletes inherit the last non-empty workout group value from above
  • You can have rows with just a workout group name to set the group for following athletes

Example:

First Name,Last Name,Grade,Team,Workout Group
,,,,"Sprint Group"
John,Doe,11,Varsity,
Jane,Smith,12,JV,
,,,,"Distance Group"
Bob,Johnson,10,A,
Alice,Brown,11,Varsity,
